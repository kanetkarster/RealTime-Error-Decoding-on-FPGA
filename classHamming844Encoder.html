Detailed Description

Encodes a 4 bit input using the Hamming 844 formula.

Parameters
datadata to be encoded. (m0 = data(0), ... m3 = data(3))
Return values
encoded_datadata encoded with the Hamming formula

implemeted as follows: encoded_data (7->4) = data(3->0) encoded_data (3) = data(0) + data(1) + data(2) encoded_data (2) = data(0) + data(1) + data(3) encoded_data (1) = data(0) + data(2) + data(3) encoded_data (0) = data(1) + data(2) + data(3)
